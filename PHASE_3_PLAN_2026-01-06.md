### 3.4: Improve Type Safety (~1 hour)
**Issue:** Unsafe type assertions, optional chaining without proper guards

**Locations:**
1. `app/r/[tracking_code]/page.tsx:46` - `params?.tracking_code as string`
2. `app/p/[slug]/page.tsx` - Similar pattern
3. Various `data!` non-null assertions

**Strategy:**
1. Replace `as string` with proper type guards
2. Use validation functions instead of assertions
3. Add runtime checks where needed
4. Document assumptions

**Example Fix:**
```typescript
// Before:
const trackingCode = (params?.tracking_code as string) || ''

// After:
const trackingCode = typeof params?.tracking_code === 'string'
  ? params.tracking_code
  : ''

// Or with validation:
function isValidTrackingCode(code: unknown): code is string {
  return typeof code === 'string' && code.length > 0
}

const trackingCode = isValidTrackingCode(params?.tracking_code)
  ? params.tracking_code
  : ''
```

**Success Criteria:**
- No `as` type assertions in production code
- Proper type guards for all external data
- TypeScript strict mode compliance
- Runtime validation for critical paths

### 3.5: Environment Variable Validation (~30 minutes)
**Issue:** No validation that required env vars are set

**Strategy:**
1. Create `lib/env.ts` with validation
2. Check all required env vars at startup
3. Provide clear error messages
4. Document all env vars

**Implementation:**
```typescript
// lib/env.ts
const REQUIRED_ENV_VARS = [
  'NEXT_PUBLIC_SUPABASE_URL',
  'NEXT_PUBLIC_SUPABASE_ANON_KEY',
  'SUPABASE_SERVICE_ROLE_KEY',
  'UPSTASH_REDIS_REST_URL',
  'UPSTASH_REDIS_REST_TOKEN',
] as const

export function validateEnv() {
  const missing = REQUIRED_ENV_VARS.filter(
    (key) => !process.env[key]
  )

  if (missing.length > 0) {
    throw new Error(
      `Missing required environment variables:\n${missing.join('\n')}\n\nSee README.md for setup instructions.`
    )
  }
}

// Call in middleware.ts or layout.tsx
```

**Success Criteria:**
- Clear error on missing env vars
- Fails fast at startup
- Helpful error messages
- All required vars documented
